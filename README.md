<h1>Enhancing YOLOv5 using APDC and PEA</h1>

<p>
This project improves the object detection capability of YOLOv5 by integrating 
<strong>Accurate Positioning Deformable Convolution (APDC)</strong> and 
<strong>Positional Enhancement Attention (PEA)</strong> mechanisms. 
The proposed enhancement focuses on improving feature localization, spatial awareness, 
and detection accuracy for complex and small-object scenarios.
</p>

<h2>Key Features</h2>
<ul>
<li>Integration of APDC for adaptive receptive field learning</li>
<li>Positional Enhancement Attention for improved spatial feature representation</li>
<li>Enhanced detection accuracy compared to baseline YOLOv5</li>
<li>Supports standard YOLOv5 training and inference pipeline</li>
</ul>

<h2>Applications</h2>
<ul>
<li>Aerial image object detection</li>
<li>Autonomous driving</li>
<li>Surveillance systems</li>
<li>Small object detection</li>
</ul>

<h2>Technology Stack</h2>
<ul>
<li>Python</li>

<li>YOLOv5</li>
<li>Deep Learning</li>
</ul>
